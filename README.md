# Personlig Utveckling

Personlig utveckling och livshanteringssystem för Gabriel Fiedler. Täcker hälsa, personlig utveckling, kompetensutveckling och måluppfyllels. Speglar strukturen och filosofin från Manager-repositoryt men fokuserat helt på privatlivet.

## Filosofi

1. **Ärlighet framför snällhet.** Självreflektion som undviker de verkliga problemen är meningslös. Benämn det som faktiskt händer.
2. **Få saker, gjorda väl.** Max 2-3 aktiva fokusområden åt gången. Allt annat är underhåll eller väntande.
3. **System framför viljestyrka.** Bygg vanor och strukturer som gör det rätta valet till det enkla valet.
4. **Framsteg framför perfektion.** Spåra riktning, inte bara resultat. En dålig vecka är inte ett misslyckande om du lär dig av den.
5. **Regelbunden uppföljning.** Veckovis reflektion håller dig ärlig. Kvartalsvis uppföljning justerar kursen.

## Nyckeldokument

### Utvecklingsplan.md
Centralt dokument för personlig tillväxt. Innehåller:
- Nuvarande fokusområden (max 2-3)
- Varför dessa är viktiga just nu
- Konkreta beteenden att öva på
- Hur man vet om det fungerar

### Årsmål (2025.md)
Årliga mål över livsområden. Varje mål bör ha:
- Tydliga framgångskriterier
- Varför det är viktigt
- Nyckel-milstolpar
- Vad som kan hindra det

### Veckoreview
Veckovis reflektion som kopplar ihop alla områden:
- Vad hände med träning/kost/återhämtning?
- Framsteg inom utvecklingsområden?
- Vad fungerade, vad fungerade inte?
- Justeringar för nästa vecka?

## Arbetssätt

### Veckoreviews
- Använd `struktur.md` som mall
- Var specifik med vad som faktiskt hände, inte vad du planerade
- Koppla observationer till mål och utvecklingsområden
- Notera mönster över tid

### Kvartalsreviews
- Granska framsteg mot årsmål
- Justera mål om omständigheter ändrats
- Identifiera vad som fungerar vs vad som bara är på papper
- Sätt fokus för nästa kvartal

### Hälsotracking
- `Träning.md` innehåller principer och nuvarande program
- `Kost.md` innehåller riktlinjer, inte stela regler
- `Återhämtning.md` spårar sömn- och stressmönster
